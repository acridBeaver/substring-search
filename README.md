# Поиск подстроки в строке
Версия 1.0

Автор: Литвин Юрий

## Описание
Данное приложение является реализацией бенчмарка для алгоритмов
поиска подстроки в строке. В данном варианте реализованы алгоритмы:
* Bruteforce
* Хеш-функция
* Конечный автомат
* Бойера Мура

## Требования
* Python версии не ниже 3.6
